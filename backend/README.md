# Task Manager (Full Stack MERN)

A full stack task management application built using the MERN stack.

## Features

- User authentication (Login / Register)
- Kanban board (Todo, In Progress, Completed)
- Task priority levels
- Due date reminders
- Email reminders for due tasks
- Advanced search & filters
- Dashboard analytics

## Tech Stack

Frontend
- React
- Axios
- CSS

Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Nodemailer

## Installation

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev
