# 🚀 Task Manager (MERN Stack)

A full-stack task management application built using the **MERN Stack**.

Users can create, manage, and track tasks with priority levels, due dates, and Kanban board workflow.

---

## ✨ Features

- User Authentication (JWT)
- Task Creation & Deletion
- Priority Levels (Low / Medium / High)
- Due Date Tracking
- Kanban Board (Todo → In Progress → Completed)
- Advanced Search & Filters
- Email Reminder for Due Tasks
- Modern Glassmorphism UI
- Progress Tracking Dashboard

---

## 🛠 Tech Stack

### Frontend
- React
- Axios
- CSS (Glass UI)

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Nodemailer (Email Reminders)

## ⚙️ Installation

### 1️⃣ Clone the repository
git clone https://github.com/Suraj-Singh-Pal/task-manager-fullstack.git


---

### 2️⃣ Install backend dependencies
cd backend
npm install


---

### 3️⃣ Install frontend dependencies


cd ../frontend
npm install


---

### 4️⃣ Start backend


cd backend
npm run dev


---

### 5️⃣ Start frontend


cd frontend
npm run dev


---

## 🌐 Environment Variables

Create `.env` inside **backend**


MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
EMAIL_USER=your_email
EMAIL_PASS=your_app_password
